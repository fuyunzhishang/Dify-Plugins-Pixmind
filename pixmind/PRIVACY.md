# Privacy Policy

[中文](#中文) | [English](#English)

## 中文

### 数据收集

本插件**不收集**任何用户个人数据。

### API Key 使用

用户的 API Key 仅用于向 Pixmind API 发起请求，插件不会存储、上传或以任何方式共享您的 API Key。

### 生成内容

图片和视频生成内容通过 Pixmind API 处理。有关生成内容的隐私政策，请参阅 Pixmind 的隐私政策：https://www.pixmind.io

### 联系方式

如有隐私相关问题，请联系：support@pixmind.io

## English

### Data Collection

This plugin does **not** collect any personal data from users.

### API Key Usage

Your API Key is used solely to make requests to the Pixmind API. This plugin does not store, upload, or share your API Key in any way.

### Generated Content

Image and video generation content is processed through the Pixmind API. For details on how generated content is handled, please refer to Pixmind's privacy policy: https://www.pixmind.io

### Contact

For privacy-related questions, please contact: support@pixmind.io
